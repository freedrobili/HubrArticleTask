Название проекта: Парсер статей с Habr.

Описание.
Этот проект представляет собой веб-приложение для парсинга статей с сайта Habr.
Пользователи могут просматривать заголовки и содержание статей, а также информацию об авторах и тегах.

Стек технологий:
    - Python
    - Django
    - BeautifulSoup
    - HTML
    - CSS
    - JavaScript
    - MySQL

Запуск проекта.

Шаг 1. Клонирование репозитория.

git clone <URL репозитория>
cd parser_project

Шаг 2. Установка зависимостей.
Убедитесь, что у вас установлен Python. Затем выполните следующие команды для установки необходимых зависимостей:

pip install -r requirements.txt

Шаг 3: Восстановление базы данных из дампа.
Предоставлен дамп базы данных в формате MySQL. Восстановите базу данных из дампа с помощью команды:

mysql -u root -p habr_db < HUBR_DUMPDB.sql

Где HUBR_DUMPDB.sql - это файл дампа базы данных.

Шаг 4: Настройка базы данных.
Проект использует базу данных MySQL. Убедитесь, что у вас установлен и запущен MySQL Server.

В файле settings.py в папке article_site найдите блок DATABASES и измените параметры подключения к базе данных.

Шаг 5. Применение миграций.
Примените миграции для создания таблиц в базе данных:

python manage.py makemigrations
python manage.py migrate

Шаг 6. Запуск сервера.
Запустите сервер Django с помощью следующей команды:

python manage.py runserver

После этого сервер будет запущен на локальной машине по адресу http://127.0.0.1:8000/.

Шаг 7. Просмотр приложения
Откройте веб-браузер и перейдите по адресу http://127.0.0.1:8000/ для доступа к приложению.
Вы увидите главную страницу со списком статей с сайта Habr.
